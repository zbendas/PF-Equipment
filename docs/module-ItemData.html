Type Definitions

ItemObject

Object describing an item.

An item object, containing various properties describing the weapon itself. Many of these properties are not required, so as to allow this particular object to be flexible.
Type:
  • Object
Properties:
Name Type Attributes Description
item_type string Item category. This will usually be one of the following: Unarmed, Light, One-handed, Two-handed, Ranged, or Ammunition
classification string <optional>
 Simple, Martial, Exotic, etc; Optional where item_type = "Ammunition"
name string Item name, as listed in PRD or other reference documentation.
alt_name string <optional>
 Alternate version of name, if applicable. E.g., "Flail, heavy" and "Heavy flail"
cost string <optional>
 Item cost, as listed. E.g., "2 sp", "10 gp"
amount number <optional>
 Amount of item purchasable per cost
damage_small string <optional>
 Damage dealt when wielded by Small character
damage_medium string <optional>
 Damage dealt when wielded by Medium character
critical_range string <optional>
 Critical threat range
critical_multiplier number | Array.<number> <optional>
 Critical damage multiplier
range number <optional>
 Range of weapon, as a number. 'ft.' will be appended where appropriate.
weight number | string <optional>
 Weight of weapon, as a number. 'lb.' or 'lbs.' will be appended where appropriate. In cases where weight is "special", this property can be supplied as a string.
damage_type string <optional>
 String describing the type of damage dealt. Should follow Pathfinder's standard triad of "B", "P", "S", or combinations thereof
special Array.<string> <optional>
 Array containing any special qualities of the weapon
see_text string <optional>
 If the ItemObject.special property contains "see text", this must be provided. This will store the specific extra gameplay information about the item.
